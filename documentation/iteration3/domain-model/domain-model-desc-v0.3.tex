\begin{itemize}
    \item \textbf{Πτήσεις (Flights)}: Οντότητα που αναπαριστά τις πτήσεις. Συνδέεται με αεροσκάφη και διακρίνεται σε εμπορικές και εμπορευματικές πτήσεις.
    \item \textbf{Αεροσκάφος (Aircraft)}: Οντότητα που αναφέρεται στα αεροσκάφη που εκτελούν πτήσεις. Κάθε αεροσκάφος σχετίζεται με αναφορές και πτήσεις.
    \item \textbf{Αναφορά (Report)}: Πληροφορίες που καταγράφονται σχετικά με ένα αεροσκάφος ή μια πτήση.
    \item \textbf{Εμπορική Πτήση (Commercial)}: Υποκατηγορία πτήσης που μεταφέρει επιβάτες και συνδέεται με ένα μενού φαγητού.
    \item \textbf{Εμπορευματική Πτήση (Freight)}: Υποκατηγορία πτήσης που μεταφέρει εμπορεύματα.
    \item \textbf{Μενού Φαγητού (Food Menu)}: Οντότητα που περιλαμβάνει τις διαθέσιμες επιλογές γευμάτων στις εμπορικές πτήσεις.
    \item \textbf{Φαγητό (Food Item)}: Μια μεμονωμένη επιλογή φαγητού που περιλαμβάνεται στο μενού φαγητού.
    \item \textbf{Πλήρωμα Πτήσης (Flight Crew)}: Ομάδα εργαζομένων που συμμετέχουν στη λειτουργία μιας πτήσης, συμπεριλαμβανομένων πιλότων και αεροσυνοδών.
    \item \textbf{Πιλότος (Pilot)}: Μέλος του πληρώματος πτήσης που είναι υπεύθυνο για τη διακυβέρνηση του αεροσκάφους. Διακρίνονται σε καπετάνιους και συγκυβερνήτες.
    \item \textbf{Κυβερνήτης (Captain)}: Ανώτερος πιλότος που έχει την κύρια ευθύνη για την πτήση.
    \item \textbf{Συγκυβερνήτης (Co-pilot)}: Βοηθός πιλότου που υποστηρίζει τον καπετάνιο κατά τη διάρκεια της πτήσης.
    \item \textbf{Αεροσυνοδός (Flight Attendant)}: Μέλος του πληρώματος πτήσης που είναι υπεύθυνο για την εξυπηρέτηση των επιβατών.
    \item \textbf{Επικεφαλής Αεροσυνοδός (Lead Flight Attendant)}: Ανώτερος αεροσυνοδός που έχει κύρια ευθύνη για την καμπίνα.
    \item \textbf{Πλήρωμα Εδάφους (Ground Crew)}: Ομάδα εργαζομένων που υποστηρίζουν τις πτήσεις από το έδαφος.
    \item \textbf{Μηχανικός Εδάφους (Ground Engineer)}: Μέλος του πληρώματος εδάφους που είναι υπεύθυνο για τη συντήρηση και τον έλεγχο των αεροσκαφών.
    \item \textbf{Υπάλληλος (Employee)}: Γενική οντότητα που περιλαμβάνει όλους τους εργαζόμενους, τόσο στο πλήρωμα πτήσης όσο και στο πλήρωμα εδάφους.
    \item \textbf{Αεροδρόμια (Airports)}: Οντότητα που αναπαριστά τις τοποθεσίες όπου πραγματοποιούνται οι πτήσεις και σχετίζεται με τους υπαλλήλους.
    \item \textbf{Συμβασιούχος (Contractor)}: Οντότητα που αναφέρεται σε εξωτερικούς συνεργάτες που παρέχουν υπηρεσίες σχετικές με πτήσεις ή αεροδρόμια.
    \item \textbf{Συμβασιούχος Αεροδρομίου (Airport Contractor)}: Συμβασιούχος που εργάζεται για λογαριασμό του αεροδρομίου.
    \item \textbf{Συμβασιούχος Πτήσης (Flight Contractor)}: Συμβασιούχος που παρέχει υπηρεσίες που σχετίζονται με τις πτήσεις.
    \item \textbf{Τιμολόγιο (Contracted Job)}: Έγγραφο που σχετίζεται με τις υπηρεσίες των συμβασιούχων και την οικονομική τους διαχείριση.
    \item \textbf{Επιβάτης (Passenger)}: Είναι ο επιβάτης ο οποίος αγοράζει το εισητήριο για την πτήση
    \item \textbf{Αγορά Εισητηρίου (Flight Booking)}: Περιγράφει μια οντότητα που αποθηκεύει πληροφορίες για την αγορά των εισητηρίων που έγιναν από τους επιβάτες στο γκισέ του αεροδρομίου
    \item \textbf{εντολή συντήρησης (Maintenance Order)}: Περιγράφει μια οντότητα που αποθηκεύει πληροφορίες για την συντήρηση του αεροπλάνου
\end{itemize}
